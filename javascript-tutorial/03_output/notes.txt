- Kita dapat menampilkan data menggunakan Javascript melalui beberapa cara
  1. Menulis langsung kedalam element HTML menggunakan innerHTML
  2. Menulis langsung kedalam HTML output menggunakan document.write()
  3. Menggunakan alert box seperti document.alert()
  4. Menggunakan browse console atau console.log
